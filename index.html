我的第一篇博客文章

来自(Bilibili)手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo

#1.部署环境
①安装nodejs
sudo apt-get install nodejs

②安装npm
sudo apt-get install npm

③安装cnpm
npm install -g cnpm --registry=http://registry.npm.taobao.org

④正式安装hexo
cnpm install -g hexo-cli

#2.初始化个人博客
①在/home/jian创建blog目录
mkdir blog

②初始化博客
sudo hexo init
//问题：此处一直出现throw new TypeError(‘Console expects a writable stream instance’);错误，是因为node的版本过低，要在10.0以上，更新方法如下：
a.先安装n模块
sudo npm install -g n
b.更新 node
sudo n latest

③启动博客
hexo s

#3.如何写博客
①创建博客
hexo n "我的第一篇博客文章"
②修改文章内容
vim 我的第一篇博客文章 //到相应目录下
